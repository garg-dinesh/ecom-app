# ecom-app

How to Setup the Project?

1. Get a clone of the project.
2. Backend (server) setup:
      go to the server directory and follow the below steps:
      -> npm install
      -> npm i -g json-server
      -> json-server --watch db.json
3. Client Setup:
     go to test app and then follow the below steps:
     -> npm install
     -> npm start
